# 🎴 Interactive Card Hover Effect

A simple and responsive card navigation project built using **HTML** and **CSS**. The project contains **4 stacked cards**. When the user hovers over the card, the top card moves to reveal the other cards hidden behind it, creating a smooth and attractive hover animation.

## ✨ Features

- 🎴 4 stacked cards
- 🖱️ Smooth hover animation
- 🎨 Built using only HTML and CSS
- 📱 Responsive design
- ⚡ No JavaScript required
- 💻 Clean and simple code

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
Cards/
```

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Krishna-gupta73/Cards.git
   ```

2. Open the project folder.

3. Open `cards.html` in your web browser.

## 📸 Preview

The project displays four stacked cards. When you hover over the card, the top card slides to reveal the other hidden cards with a smooth animation.
